#### ClockWindow
2021年11月16日  
上传了一个小工具，在MyDock中显示时间，效果如图。  
因为升级到Win11后任务栏难用丑陋，所以就用MyDock替换了任务栏，但是个人不喜欢开启MyFinder，导致没有查看时间的地方，加上MyDock不开源因此自己写了这个小工具。
